# Hermes Skill → Crazor 数字员工映射

> 整理日期：2025-05-23
> 基于 Hermes Agent 108 个 Skill，按岗位拆分为 11 个数字员工。

---

## 分级标准

- **A 级**：可独立成为数字员工 — 技能闭环，端到端业务价值
- **B 级**：可并入某个数字员工 — 职能聚焦但不够闭环，作为某员工的子能力
- **C 级**：仅作底层 Skill — 太窄/基础设施/纯个人用途，不具备独立业务价值
